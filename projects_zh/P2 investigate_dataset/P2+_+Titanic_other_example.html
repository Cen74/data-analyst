## Data cleaning, use full name of ports instead of code, to make it easier to read the data without extra notes.

def convert_port(port_code):
    if port_code == 'C':
        return  'Cherbourg'
    elif port_code == 'Q':
        return  'Queenstown'
    else:
        return  'Southampton'
    
def convert_class(class_code):
    if class_code == 1:
        return  '1st'
    elif class_code == 2:
        return  '2nd'
    else:
        return  '3rd'

def correlation(x, y):
    std_x = (x - x.mean()) / x.std(ddof=0)
    std_y = (y - y.mean()) / y.std(ddof=0)
    return (std_x * std_y).mean()

Conclusion¶

Limitation of the conlusion:¶

  • Because the data only include 891 out of the 2224 passengers, and the sampling methon is unknown, the observation of the sample may not reflect the situation of Titanic incident precisely. Also, the correlations state below represent the situiation of the sample only, not tested toward the conclusion of causation.

What is the distribution of ticket price? The dataset may provide a good example of price level at that time.¶

  • The fare range from 0 to 512.33, with sd of 49.63. At least 50% of the pax brought the ticket within the range approx from 10 to 50, some tickets are more expensive, but extremely rare for tickets whose price exceeding 70. Average price for higher class is higher, and 1st class have widest range, all the classes have a tail in higher price end.

Given the available information (Class, Gender, Age etc.) in the data set, Which factor may have more correlation to the survival ratio than others?¶

  • From the correlation calculation, survival ratio seems not related to the age of passengers (r = -0.07), the number of spouse on board (r= -0.03), the number of parent and children on board (r = 0.08). However, passengers with 5 or more spouse on board did not make themself survive.
